#hoard 
Migrate all js and css dependencies within an html document from remote to local.

<b>Usage</b>: python hoard.py \<html_target\> \<resources_directory\>

<b>Requirements</b>
  <br>  beautifulsoup4==4.4.1
  <br>  bs4==1.2.5
  <br>  colorama==0.3.6
  <br>  requests==2.9.1
  <br>  wheel==0.24.0
